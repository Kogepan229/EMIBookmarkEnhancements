EMIBookmarkEnhancements is an EMI mod that implements the Crafting Chain feature of the bookmark panel from GTNH's NotEnoughItems (NEI), based on the LGPL-3.0 licensed project JEI-Enhancements. This mod was created entirely by an AI coding agent.

Reference projects:
- JEI-Enhancements: https://github.com/GaLicn/JEI-Enhancements
- GTNH NotEnoughItems (NEI): https://github.com/GTNewHorizons/NotEnoughItems

Special thanks to the original author of JEI-Enhancements, GaLicn, and the GTNH team.
